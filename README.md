# Boxing Timer

A lightweight interval timer for boxing, calisthenics, and other round-based workouts.

**https://timer.crcibernetica.com**

## Features
- Configurable rounds, round duration, rest, prep countdown, and warning alert
- Built-in presets (Boxing, Calisthenics)
- Custom bell and warning sound effects
- Mute toggle
- Responsive layout (mobile portrait, landscape, and desktop)
- Installable as a PWA with offline support
- Screen wake lock during workouts

## Usage
Just open the URL in any modern browser. No install required — or add it to your home screen for a native app experience.
